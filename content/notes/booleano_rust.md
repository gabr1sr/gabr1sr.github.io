É um valor que pode ser somente `true` ou `false`.


## Declaração {#declaração}

Podemos declarar das seguintes formas:

```rust
let bool1: bool = true;
let bool2: bool = false;

println!("bool1 = {:?}", bool1);
println!("bool2 = {:?}", bool2);
```
